
# Pushoong-ish 익명 Q&A (다대일)
PUSHOONG을 **참고한 카드형 UI**이지만, **원본 소스 복제 없이** 새로 구현한 공개 Q&A 피드입니다.

- 공개 Q&A: `/@slug` (카드 피드, 좋아요/신고(클라 시뮬레이션))
- 소유자 콘솔: `/owner/slug` (로그인 후 답변 발행)
- API/서버 구조: Express + Socket.IO (파일 저장)

## 실행
npm install
npm start
→ http://localhost:3000

## 배포(Render)
Build: npm install
Start: npm start
